MicroPython for Surenoo Guition W5 Round

Posted on Sat 05 July 2025 in Board

Follow the steps below to install MicroPython on your Surenoo Guition W5 Round ESP32S3 board from your browser. You can review or rebuild the source code using the project repo.

  • Connect board to PC using USB-C cable
  • Press the Connect button below and follow prompts (use serial port USB JTAG/serial debug unit)

Your browser does not support WebSerial You are not allowed to use this on HTTP

Web installation is thanks to the ESP Web Tools project.
